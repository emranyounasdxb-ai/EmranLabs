# EmranLabs

Official source repository for **EmranLabs**, a technology brand focused on modern software products, AI solutions, SaaS platforms, web applications, mobile applications, and business automation.

## Current status

The repository has been initialized and connected for ongoing development.

## Included prototype

- `prototypes/windows-2030/index.html` — standalone interactive desktop UI concept built with HTML, CSS, and vanilla JavaScript.

## Recommended production stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase / PostgreSQL

The Windows 2030 prototype is stored as a design and interaction reference. It is not yet the production EmranLabs website.
